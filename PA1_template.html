2. Devise a strategy for filling in all of the missing values in the dataset.

3. Create a new dataset that is equal to the original dataset but with the missing data filled in.

The strategy used is to impute the missing values with the corresponding average steps calculated previously for that specific five minute time interval.

stepdataimputed <- transform(stepdata, 
                    steps = ifelse(is.na(stepdata$steps),
                    avgsteps$averagesteps[match(stepdata$interval,avgsteps$interval)],
                    stepdata$steps))
